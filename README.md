# Beginner Laravel

低スペックの PC でも触りやすい、Laravel 入門用の開発環境です。

## 起動方法

ターミナルでこのフォルダを開いて、次のコマンドを実行します。

まず、`setup.sh` を実行できるようにします。

```bash
chmod +x setup.sh
```

そのあと、セットアップを起動します。

```bash
./setup.sh
```

自動で次のことを行います。

- `.env` がなければ作成
- SQLite データベースを作成
- Laravel のアプリキーを作成
- データベースの初期設定
- ブラウザで `http://127.0.0.1:8000` を開く
- Laravel の開発サーバーを起動

終了するときは、ターミナルで `Ctrl + C` を押してください。

## ポートを変えたいとき

8000 番ポートが使われている場合は、別の番号で起動できます。

```bash
PORT=8001 ./setup.sh
```

## この環境の方針

最初は軽く動かすため、Vite や npm の開発サーバーは起動していません。
画面を表示して Laravel を学ぶだけなら、`./setup.sh` だけで十分です。

JavaScript や CSS のビルドが必要になったら、別途 npm の設定を使います。
